HERA Microprocessor

The HERA (Haverford Educational RISC Architecture) Microprocessor is composed of: ALSU (Arithmetic/Logic/Shift Unit), Operation control, Register file, Program ROM, PC register and Flags handler. The user inputs HERA machine code and run a simulation. The microprocessor steps through each component depending on the machine code and outputs the values stored in the register file. I implemented the HERA microprocessor in Logism. The circuit is available here.

Ising Model

The Ising model allows to study the magnetization of a system using dipole moments of spins. Each spin points either up or down depending on its magnetic moment. To observe the behavior of the Ising model within different systems, I implement the Monte Carlo method in Python which relies on randomness to calculate the magnetization of a system. The Monte Carlo simulation is computationally expensive. For each temperature, I iterate over 10000 steps to calculate the magnetization and the energy. To make my code run faster, I tried to avoid unnecessarily re-calculating previously calculated values. For more details, check out my Github repository.
